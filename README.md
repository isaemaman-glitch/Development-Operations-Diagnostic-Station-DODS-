# Development-Operations-Diagnostic-Station-DODS-
Development, Operations  & Diagnostic Station (DODS)

DODS is a Raspberry Pi Pico based development and troubleshooting platform designed to assist with electronics prototyping, testing, calibration, and diagnostics.

## Need To Know 
this is adrduino ide based code and all liberys needed in arduino are in the #include section of code
## Current Features

- Servo tester
- ESC tester
- PWM signal generation
- Potentiometer input control
- Center-position mode
- Status LEDs
- Serial diagnostics

## Planned Features

- DC motor testing
- ADC monitoring
- Multi-tool operating modes
- Sensor diagnostics
- Display interface
- Data logging

## Hardware

- Raspberry Pi Pico
- SG90 Servo
- 10K Potentiometer
- Push Button
- RGB LEDs
